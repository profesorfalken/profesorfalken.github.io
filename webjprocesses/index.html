Get all the processes information and manage them

Works on Windows, Mac and Linux!

jProcesses exposes a very simple API to retrieve the list of running processes and manage it (kill, give priority, ...).

Version 1.1

Version 1.0 ready

Feature complete version that also allows to filter by PID or name and to change priority.

Version 0.2 ready

All the code is refactored to use 100% native Java code instead of JNI bindings to C code.
